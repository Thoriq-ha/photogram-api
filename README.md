## [Project] Photogram
API for Photogram

### Requirement:

- PHP >= 7.4
- Laravel 8.x
- Composer 2.x

### Install

- Clone this repository
- Run `composer install`
- Run `cp .env.example .env`
- Run `php artisan key:generate`
- Settings `.env` file

### Database Migration and Seeder

- Run `php artisan migrate --seed`
